Add .env file!

```shell
WIFI_PASS=
WIFI_SSID=
SERVER_ADDR=
```
SEVER_ADDR is the adress of the senec server inside your local network


Build on Linux/MacOS ( check dependencies of espflash etc.):
```shell
cargo run 

```
Build on Windows using wsl ( again check the dependencies of espflash etc.):
```shell
sh w.sh
```
